# daily-algo-mem
Implementing a random algorithm everyday for 30 days to understand them better.

I'm using [this](https://github.com/catfoodeater/random-choice) to pick an algorithm.

| Day | Algorithm | Complete | Time |
|---|---|---|---|
| 1 | Binary search | ✅ | N/a |
| 2 | Bubble sort | ✅ | <5m |
| 3 | Merge sort | ✅ | >30m |
